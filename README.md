# RetailFlow Capstone Project

## Project Overview
This project demonstrates a cloud-based data pipeline using AWS services.

## Technologies Used
- Python
- AWS S3
- Boto3
- Google Colab
- VS Code

## Project Structure
- 01_data_generation.ipynb - Generates synthetic retail data.
- Phase 1 - Synthetic Data Generation & Profiling
- Phase 2 - Boto3 Ingestion Utility
- Phase 3 - S3 Data Lake Foundations

## How to Run
1. Generate synthetic data using the notebook.
2. Configure AWS credentials.
3. Run the Boto3 upload script.
4. Verify files in the S3 bucket.
